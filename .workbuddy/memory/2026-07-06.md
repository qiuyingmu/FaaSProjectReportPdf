## PDF 导出（Open HTML to PDF 替换 Flying Saucer）

`flying-saucer-pdf:9.1.22` → `openhtmltopdf-pdfbox:1.0.10`

**为什么换**：Flying Saucer (iText 2.1.7) 只支持 CSS 2.1，不认识 flexbox/grid/gradient/border-radius/box-shadow/pseudo-elements，PDF 渲染效果差。

**Open HTML to PDF 优势**（基于 Apache PDFBox 2.0.24）：
- 支持 CSS 3：border-radius、linear-gradient、box-shadow、伪元素 ✅
- 支持 flexbox 基本布局 ✅
- 更好的 table 渲染 ✅
- 可嵌入中文字体（自动检测 simhei.ttf / msyh.ttc / simsun.ttc）

**改动**：
- `ReportPdfExporter.java` 重写：`PdfRendererBuilder` 替代 `ITextRenderer`，支持中文自动检测+嵌入
- 现在直接使用 `ReportHtmlBuilder.build()` 的完整 HTML（含所有现代 CSS 样式）
- `ReportPdfBuilder.java` 保留但不再被调用（作为 fallback）
